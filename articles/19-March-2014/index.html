这是 Javascript 系列文章的第五篇，已经连续写了五天，Javascript 学习笔记暂时告一段落。今天讲讲怎样用 Javascript 来做比较。

程序里面经常会涉及到两个值的比较，Javascript 默认按照字母顺序进行比较，即使输入的是数字，Javascript 也会先自动把它们转化成为字符，然后进行比较。所以，当比较的对象不是字符时，会出现意想不到的结果。比如：2 与 10 比较，结果是 10 小于 2，因为 10 的第一个字符 1 小于 2。

为了解决这个问题，Javascript 提供了一个方法 sort()，它让程序员来定义比较大小的方式。经过比较之后，传递一个值给 sort()，这个值可以是正、负或者零。

比较两个数 a 和 b 的方法非常简单：

function cmp(a,b) {
    return a - b;
}

将这个方法作为 sort(cmp) 的参数：

  1. 如果 a 小于 b，返回值为负值，那么 sort(cmp) 方法会把 a 排在 b 的前面。
  2. 如果 b 小于 a，返回值为正值，那么 sort(cmp) 方法会把 b 排在 a 的前面。
  3. 如果 a 等于 b，返回值为零，那么 sort(cmp) 会认为两个值相等。

下面是一个具体的例子：

<!DOCTYPE html>
<html>
<head>
    <title>列表自动排序</title>
</head>
<body>
    <h1>列表自动排序</h1>
    <p>输入数字，点击确定，浏览器会将输入过的数字进行 <i>从小到大</i> 排序。</p>
    <form>
        请输入一个整数：<input type="text" id="input" />
        <input type="button" id="submit" value="确定">
    </form>
    <p>输出：</p>
    <p id="output"></p>

    <script type="text/javascript" >
        var theList = new Array()

        function compare(buttonId, input, output) {
            var btn = document.getElementById(buttonId)
            btn.onclick = function() {
                var inputNumber = document.getElementById("input").value
                theList.push(parseInt(inputNumber))

                if (1 < theList.length) {
                    theList = theList.sort(function(a,b) {
                        return a - b;
                    })
                }

                document.getElementById(output).innerHTML = theList
            }
        }

        var m = new compare('submit', 'input', 'output')
    </script>
</body>
</html>
